Отчет к домашнему заданию №3
1. Условие задачи:
В рамках выполнения данного домашнего задания требуется разработать программу строкового калькулятора. Причём по запросу с клавиатуры в консольном приложении вводится строка, которая может содержать:

знаки операций +,-,*,/;
константы (целые или вещественные);
скобки ();
функции sin, cos, tg, ctg, exp;
переменную x.
Если переменная x есть в формуле, то запрашивается её значение, после чего вычисляется результат и выводится на экран.

2. Схема алгоритма
Программа строкового калькулятора работает по следующему алгоритму:

  а) запись исходной формулы в постфиксной форме;
  б) определение наличия переменной x в данной формуле и запрашивание её значения;
  в) вычисление значения данного выражения при помощи его записи в постфиксной форме;
  г) вывод значения на экран пользователя.
